APC UPS Network Card Compatibility:

The following website is the official APC compatibility list: APC UPS Networking Management Card

Note that even old devices support most forms of SmartSlot cards. See the following link for a list of SmartSlot cards on Wikipedia

(Please note that not all of these cards are IP networking cards, please check their descriptions)
